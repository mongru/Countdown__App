## Countdown__App
A simple countdown app built with React

## Installing
For local changes and edits clone and download this repo.

Then run:

```
npm install
```

and:

```
npm start
```

& have fun editing!

## Built With
* ReactJS + ES6
* [React-Bootstrap](https://react-bootstrap.github.io/)
* Webpack
